Dimensionamento de Moinho utilizando o Método Bond

Desde meados dos anos 50, o método Bond vem sendo utilizado para o dimensionamento de moinhos, tal método é baseado no ensaio de Wi e é aplicavel, principalmente, em circuitos tradicionais. Mesmo sendo bastante utilizado, tal método apresenta diversas limitações.

Diante disso foi solicitado pelo professor José Yvan Pereira Leite(in memoriam) o desenvolvimento desse sistema para o uso em experiencias laboratoriais, visto que alguns calculos são consideravelmente complexos caso sejam feito sem algum tipo de auxilio de calculadoras ou computadores.
